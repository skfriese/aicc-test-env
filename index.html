AICC Launch Configuration

Enter the full URL to the AU and click the Launch Content button below.
Modify this value to specify the AICC_SID (session ID) parameter, which will be passed to the content when launched.
Modify this value to specify the AICC_URL (LMS URL) parameter, which will be passed to the content when launched. To allow this test environment to act as the LMS and capture communication, leave this value as-is.
Add parameters which are specific to the AU (in format "param1=value1&param2=value2...").
